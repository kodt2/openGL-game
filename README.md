# openGL-game

Небольшой 3D-проект на **C# + OpenTK**, в котором отрисовывается процедурная сцена из кубов: ландшафт, деревья, дорога и автомобиль с управлением.

## Что уже реализовано
- Рендеринг 3D-сцены на OpenGL.
- Базовые шейдеры для цветной и текстурной геометрии.
- Генерация деревьев и теней.
- Генерация дороги и размещение объектов по карте.
- Управление автомобилем и камерой.

## Управление
- `W` — газ
- `S` — тормоз / движение назад
- `A` / `D` — поворот
- Движение мыши — вращение камеры
- `Esc` — выход

## Структура (основное)
- `Program.cs` — инициализация окна, шейдеров, рендера и игрового цикла.
- `Class1.cs` — базовые объекты сцены (кубы, плоскости, деревья и т.д.).
- `texture.jpg` — текстура, используемая в сцене.
