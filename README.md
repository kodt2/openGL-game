# openGL-game

Небольшой 3D-проект на **C# + OpenTK**, в котором отрисовывается процедурная сцена из кубов: ландшафт, деревья, дорога и автомобиль с управлением.

## Что уже реализовано
- Рендеринг 3D-сцены на OpenGL.
- Базовые шейдеры для цветной и текстурной геометрии.
- Генерация деревьев и теней.
- Генерация дороги и размещение объектов по карте.
- Управление автомобилем и камерой.

## Управление
- `W` — газ
- `S` — тормоз / движение назад
- `A` / `D` — поворот
- Движение мыши — вращение камеры
- `Esc` — выход

## Структура (основное)
- `Program.cs` — инициализация окна, шейдеров, рендера и игрового цикла.
- `Class1.cs` — базовые объекты сцены (кубы, плоскости, деревья и т.д.).
- `texture.jpg` — текстура, используемая в сцене.

## Что было улучшено в рефакторинге
- Убрано дублирование логики создания шейдерных программ.
- Переименованы методы компиляции шейдеров для лучшей читаемости.
- Убран жёстко захардкоженный абсолютный путь к текстуре, добавлен безопасный поиск `texture.jpg`.

## Запуск
Поскольку в репозитории сейчас нет `.csproj`/`.sln`, запуск зависит от того, как ты организуешь проект локально.

Минимальный вариант:
1. Создай новый `console` проект на .NET.
2. Добавь `Program.cs`, `Class1.cs` и `texture.jpg`.
3. Установи зависимости (`OpenTK`, `SixLabors.ImageSharp`).
4. Запусти проект через `dotnet run`.
